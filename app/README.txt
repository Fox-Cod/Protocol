ЧТО ЭТО:
Настоящее десктоп-приложение (Electron) — своё окно,
своя иконка в панели задач, без браузера и без VBS.
Работает полностью офлайн.

ЧТО НУЖНО (один раз):
1. Установить Node.js:  https://nodejs.org  (LTS-версия)

СБОРКА (в этой папке):
1. Открой командную строку в этой папке
   (Shift + ПКМ в папке → "Открыть окно PowerShell здесь")
2. Введи:   npm install
   (скачает Electron — подожди пару минут)
3. Введи:   npm run dist
   (соберёт приложение)

РЕЗУЛЬТАТ:
В папке dist появится файл  Protocol.exe
Это и есть готовое приложение — запускай двойным кликом,
копируй куда угодно, работает без установки.

ЗАПУСК БЕЗ СБОРКИ (для проверки):
   npm install
   npm start
