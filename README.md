# Protocol

**Быстрый справочник по законам Majestic RP — прямо поверх игры.**

Уголовный, Гражданский, Процессуальный и другие кодексы, разбор ситуаций «кто прав и по какой статье», защита от вопросов адвоката на допросе, заметки-стикеры поверх игры и умный поиск — всё в одном лёгком десктопном приложении.

## 🚀 Установка

Никакой сборки и настройки — просто скачай и запусти.

1. Открой страницу **[Releases](../../releases/latest)**
2. Скачай файл **`Protocol.exe`**
3. Запусти — всё, приложение готово к работе

> Portable-версия: не требует установки, не оставляет следов в системе. Просто держи `.exe` в удобной папке.

### 🎙️ Процессуальное действие — защита на допросе
Вбей статью, которую вменяют, и получи готовые ответы на каверзные вопросы адвоката: что спросят, что отвечать, где ловушка. Разбор по составу преступления, общие вопросы, отдельный блок «коп против копа».

### ⚖️ Ситуация — кто прав и по какой статье
Опиши ситуацию своими словами («сняли маску», «обыск», «оружие») — умный поиск покажет обе стороны сразу: на что вправе опереться коп, какие права у гражданина и какую статью нарушает тот, кто не прав.

### 🎮 Режим фокуса игры
Приложение может висеть поверх игры, **не забирая фокус** — управление и звук остаются в игре, а шпаргалка всегда под рукой. Идеально для оперативной работы во время РП.

### 🔍 Умный поиск
- **Глобальный поиск** по всем кодексам сразу — из боковой панели
- **Поиск по ситуации** понимает живую речь и синонимы
- **Поиск по вопросам** адвоката в режиме допроса
- Поиск по номеру статьи и по смыслу

## ⌨️ Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| `Ctrl + F` | Открыть «Проц. действие» и поиск по вопросам адвоката |
| `Ctrl + D` | Включить / выключить режим фокуса игры |
| `Ctrl + B` | Открыть избранное |
| `Ctrl + 1…9` | Быстрый переход между кодексами |
| `Ctrl + Shift + P` | Аварийно вернуть окно, если оно потерялось |
| `Esc` | Очистить поиск / закрыть панель |

<br/>

## 🖥️ Управление окном

- **Перетаскивание** — за верхнюю полоску окна (как у заметок)
- **Поверх других окон** — кнопка ⇧ в шапке
- **Сворачивание / закрытие** — стандартные кнопки справа
- **Сворачивание меню** — стрелка ◀ рядом с логотипом

<br/>

## 🌐 О правилах

Приложение построено на правилах сервера **Orlando** (Majestic RP). В боковой панели есть переключатель серверов — остальные серверы появятся в будущих обновлениях со своими правилами.

> Правила на сервере иногда обновляются. Перед тем как ссылаться на статью в важной РП-ситуации, сверься с актуальным законодательством на форуме или в правилах сервера.
