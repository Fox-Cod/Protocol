## ✨ Возможности

- **📚 Кодексы** — полные тексты: Уголовный, Гражданский, Этический, Процессуальный, Административный кодексы, правила гос. структур, законы об оружии. Умный поиск по номеру статьи и по смыслу.
- **⚖️ Ситуация** — разбор частых стычек «коп ↔ гражданин» (задержание, обыск, применение силы, проверка документов и др.). Показывает, что **требует коп**, что **обязан гражданин** и **чего коп НЕ вправе** — со ссылками на статьи.
- **🔨 Процессуальное действие** — «игра с адвокатом»: вводишь статью, которую вменил, и видишь каверзные вопросы адвоката (с терминами из ПК), краткие ответы офицера и где ловушка. Отдельный раздел **«Коп против копа»** — защита от наглого коллеги. Есть глоссарий терминов (что такое «субъект», «умысел» и т.д.).
- **📌 Стикеры-заметки** — закрепляй нужные статьи и ответы отдельными окошками поверх игры. Редактируемые, с регулировкой прозрачности и цвета текста.
- **🎮 Игровой режим** — окно не крадёт фокус у игры: звук не пропадает, управление остаётся в игре. Синяя рамка показывает, что режим активен.
- **⌨️ Глобальные горячие клавиши** — вызывай приложение прямо из игры: нажал клавишу → окно всплыло, курсор уже в поиске → печатаешь → `Esc` → снова в игре.
- **⭐ Избранное и калькулятор срока** — сохраняй частые статьи, считай суммарный срок и штраф по делу.

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl + F` | Открыть «Процессуальное действие» и поставить курсор в поиск по вопросам |
| `Ctrl + D` | Включить / выключить игровой режим (фокус на игре) |
| `Ctrl + B` | Избранное |
| `Ctrl + M` | Калькулятор срока |
| `Ctrl + 1…9` | Переключение между кодексами |
| `Esc` | Очистить поиск / вернуть фокус игре |

> Если основная комбинация занята другой программой, автоматически используется запасная (`Alt + F`, `Alt + D` и т.д.).

## 🚀 Установка и запуск

### Готовый .exe (проще всего)

1. Перейди в раздел [**Releases**](../../releases).
2. Скачай `Protocol.exe`.
3. Запусти двойным кликом. Установка не требуется, интернет не нужен.

### Сборка из исходников

**Требуется [Node.js](https://nodejs.org) (версия LTS).**

```bash
# 1. Клонировать репозиторий
git clone https://github.com/USERNAME/protocol.git
cd protocol

# 2. Установить зависимости
npm install

# 3. Запустить приложение
npm start

# 4. Собрать .exe (появится в папке dist/)
npm run dist
```

Готовый файл `Protocol.exe` будет в папке `dist/`.

## 🎮 Как пользоваться

1. **Запусти Protocol**, затем сверни его и заходи в игру.
2. Поставь игру в режим **«окно без рамки» (borderless)** — иначе поверх полноэкранной игры окно не всплывёт (ограничение Windows).
3. Во время игры, когда нужен закон или ответ:
   - Нажми **`Ctrl + F`** → окно всплывёт, курсор уже в поиске.
   - Введи термин или вопрос → читай ответ.
   - Нажми **`Esc`** → фокус вернётся в игру.
4. Нужную статью или ответ можно **закрепить стикером** 📌 — он останется висеть поверх игры.

> **Про звук:** если звук игры пропадает при переключении — это настройка самой игры («Mute on focus loss» / «Приглушать звук в фоне»). Выключи её и включи режим «окно без рамки».
