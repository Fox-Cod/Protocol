# Как выложить новую версию

1. Собери `Protocol.exe`
2. Зайди в **Releases** → **Draft a new release**
3. Укажи тег версии (например `v1.27.1`)
4. Прикрепи файл `Protocol.exe`
5. Опубликуй

Кнопка «Скачать» в README ведёт на `releases/latest` — всегда на свежий релиз.

> После смены версии не забудь обновить бейдж версии в README.md
> (строка `![Версия](...Версия-1.27.1...)`).
